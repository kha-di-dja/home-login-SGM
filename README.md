# 📚 SGM - Students Grades Management System

A complete web-based system to manage student grades, subjects, and academic performance for high schools.

## 🎯 Features

- **Student Management** - Register, edit, and delete student records
- **Subject Management** - Organize courses and subjects
- **Grade Management** - Record and update student grades
- **Ranking System** - Display student rankings and averages
- **User Authentication** - Secure login system for administrators and teachers
- **Responsive Design** - Works on desktop, tablet, and mobile devices

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Icons:** Font Awesome 6
- **Server:** XAMPP / Apache
